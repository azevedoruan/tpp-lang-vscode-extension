# tpp lang syntax highlighting extension for VS Code

tpp Lang is a example language used for Compiler studies at my university. It is a Portuguese-based language and is quite limited, therefore, it is not recommended for real-cases use.

## Features

This language includes very limited structures and datatypes.

Available keywords are:

- `fim` represent the end of a scopes in other languages.
- `repita` `até` corresponds to the `repeat` `until` structure.
- `se` `então` corresponds to the `if then` structure.

Dataype just includes:

- `inteiro` similar to 32 bit `int`.
- `flutuante` similar to `float`.

## Extension Settings

Since this is just a sample language, It is not available in the VS Code extensions marketplace. You must install this project directly into your `.vscode/extension` directory.

Steps:

1. Download this Zip project or make `git clone`.
2. Paste it in the `<your_name>/.vscode/extension` directory.
3. Open the VS Code in other folder with a .tpp file
4. The code with correct syntax will be highlighting.
